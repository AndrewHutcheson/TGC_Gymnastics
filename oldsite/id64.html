Rules

TGC Men's Rules                              Updated 4/4/13

1. To see the men's FIG Code of Points, click on link below, select 03 MAG then 01-1 MAG.

2013-2016 Men's FIG Code of Points

2. The men use NCAA men's gymnastics rules and count 3 scores per event (NCAA counts 4). Basically, NCAA rules are the FIG (International) standard rules except that a B as well as a C dismount will receive 0.3 Element Group 5 credit.
NEW for 2011: NCAA FX is now timed per FIG Rules (70 sec.)
>70 sec .1 deduction
>72 sec .3
>75 sec .5
Also, Top 10 skills are counted for difficulty prior to determining special requirements (does not affect most routines).
Use all the items from this NGJA page for NCAA Rules for Men:

NCAA Rules & Supplements

3. Use all the USAG Interpretations (and Vault Table) from this NGJA page for Men:

USA Interpretations

DO NOT use JO rule intrepretations for TGC Competitions.
However, NAIGC Nationals use the JO Level 9 Rules for Men. Very similar now to FIG/NCAA except only 8 skills count rather than 10 and JO interpretations apply (such as any simple gymnastics skill receives an A value but no element group credit).
